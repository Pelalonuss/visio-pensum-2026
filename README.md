# 🚀 Visio Pensum 2026

**Interaktives, spielerisches & geiles Dashboard für Patrik's Visio Coaching Pensumsteigerungsplan**

![Visio Dashboard](https://raw.githubusercontent.com/Pelalonuss/visio-pensum-2026/main/screenshot.png)  
*(Screenshot folgt – einfach die index.html öffnen für den vollen Effekt)*

## Features

- **Jahresübersicht** mit 12 klickbaren Monats-Cards + Mini-Heatmaps
- **Interaktiver Monatskalender** (Modal) – jeden geplanten Tag abhaken
- **Gamification**: Vision XP, Level-System, Streak-Counter (speichert lokal)
- **Confetti-Explosionen** beim Abschließen von Trainingstagen
- **Heute abhaken** Button (13. Mai 2026 live markiert)
- **Motivations-Quotes** & Export-Funktion
- **Voll responsive** + Dark Mode + Swiss Touch 🇨🇭
- **100% offline** (LocalStorage)

## Datengrundlage

Der Plan basiert exakt auf deinem Original-Kalenderbild (Pensum-Werte wie 2,5h in Mai/Juni etc. übernommen & erweitert mit logischer Steigerung von 1,6h → 3,8h).

Durchschnitt 2026: **2,84 Stunden/Tag**  
Gesamt-Training: **1.032 Stunden** = 43 volle Trainingstage!

- Pure HTML + Tailwind CSS (CDN)
